# Session Log: Settings File Spurious Open Fix

**Date:** 2026-06-05  
**Duration:** Brief  
**Requestor:** Eric De Carufel

## Summary

Diagnosed and fixed issue where VS Code user profile settings.json was opening unexpectedly on NexKit startup. Root cause: RecommendedSettingsConfigDeployer.deployVscodeSettings() was called during automatic activation, writing to ConfigurationTarget.Global which triggers VS Code to open the settings file. Solution: moved user-level settings deployment out of automatic verification into explicit workspace initialization phase. Activation now only performs workspace-scoped checks. All 358 tests passing.
